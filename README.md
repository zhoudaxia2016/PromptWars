# PromptWars

## 模块

### cubic · 3D 魔方

WCA 随机打乱，支持 AI 还原提示词。

🔗 **[在线体验](https://zhoudaxia2016.github.io/PromptWars/cubic/index.html)**

- 3D 魔方交互（鼠标拖拽视角、滚轮缩放）
- WCA 打乱公式生成
- 贴纸状态展示、复制 54 字符
- 复制 AI 还原提示词，粘贴给 AI 输出还原公式
- 输入并执行公式

**本地运行：**

```bash
cd cubic
python -m http.server 8080
```

访问 http://localhost:8080

### crossword · 日语填字游戏

日语词汇填字游戏，使用罗马音输入。

🔗 **[在线体验](https://zhoudaxia2016.github.io/PromptWars/crossword/index.html)**

- 横向 / 纵向日语单词
- 罗马音输入，平假名显示
- 点击提示可定位到对应格子

**本地运行：**

```bash
cd crossword
python -m http.server 8080
```

访问 http://localhost:8080
